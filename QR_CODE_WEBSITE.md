## Cara Menggunakan QR Code

### 1. Download QR Code
- Klik kanan pada gambar QR Code di atas
- Pilih "Save image as..."
- Simpan dengan nama `qr-code-kugar.png`

### 2. Untuk Media Sosial
- **Instagram Story**: Upload gambar QR Code dengan sticker "LINK"
- **Facebook**: Share gambar QR Code di postingan
- **WhatsApp**: Kirim gambar QR Code ke grup atau kontak

### 3. Untuk Media Cetak
- **Brosur/Flyer**: Cetak QR Code di ukuran minimal 2x2 cm
- **Spanduk**: Cetak QR Code di ukuran minimal 3x3 cm
- **Kartu Nama**: Tambahkan QR Code di belakang kartu nama

### 4. Untuk Website
- Tambahkan QR Code di halaman kontak
- Embed di footer website
- Gunakan di halaman "Tentang Kami"
